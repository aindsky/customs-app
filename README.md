# 报关单填充工具

自动解析订单信息并生成报关单的桌面应用程序。

## 功能特点

- 📋 自动解析订单信息（订单号、收件人、地址等）
- 📦 支持6种常见商品类型选择
- 💾 支持导出CSV格式报关单
- 🎨 简洁美观的图形界面
- 🚀 单文件EXE，无需安装Python

## 下载

前往 [Releases](../../releases) 页面下载最新版本的EXE文件。

## 使用方法

1. 下载 `报关单填充工具.exe`
2. 双击运行（如被杀毒软件拦截，请添加信任）
3. 粘贴订单信息
4. 选择商品类型
5. 点击"解析订单"
6. 可选择导出CSV文件
